# Space Pointer using Raspberry Pi and Arduino Uno
A device which can point at the position of the sun, moon and planets. The user can choose whether he wants the current position or a specific timestamp. Parts are 3D-printed.

Required parts:
- 1x Raspberry Pi (I used 3B)
- 1x Arduino (I used Uno Rev 3)
- 2x NEMA 17 Steppers
- 1x LCD (+ Keypad)
- 1x Laser 303 (or any other Laser, just modify Laser_Bracket.stl)
- 1x GPS Module
- 1x Slip Ring (I used Adafruit 22mm 12 cabels)
- 2x Bearings Outer 16mm Inner 5mm

and some screws, batteries and cabels.
